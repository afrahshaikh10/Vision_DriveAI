# 🚗 VisionDrive AI
### AI-Powered Gesture Controlled Racing Simulator

VisionDrive AI is an advanced desktop application that transforms your webcam into a virtual racing controller using Computer Vision and Artificial Intelligence.

Instead of using a physical steering wheel, users control racing games through natural hand gestures detected in real-time with MediaPipe and OpenCV.

Designed with a futuristic gaming dashboard, VisionDrive AI provides an immersive driving experience while demonstrating the practical application of AI, Computer Vision, Human-Computer Interaction, and Desktop Software Engineering.

# How It Works

1. Webcam captures live video.
2. MediaPipe detects both hands.
3. Hand landmarks are extracted.
4. Steering angle is calculated.
5. Gestures are recognized.
6. Keyboard inputs are generated.
7. Racing game receives keyboard controls.
8. Dashboard displays live analytics.
